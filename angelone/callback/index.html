Angel One callback placeholder

This page exists only to provide a public HTTPS redirect URL for the local AlgoTradingV2 SmartAPI setup.

No token, code, state, verifier, or query value is stored, displayed, forwarded, or processed by this page.

Use this redirect URL in SmartAPI: https://atuldc.github.io/angelone/callback/

AlgoTradingV2 currently uses API key, PIN, and TOTP login. It does not consume browser redirect tokens.

Do not use this static page for production OAuth secrets. GitHub Pages may receive the initial request URL in normal service logs.
